## 🎇 如何使用

### 1. 浏览分支

本仓库采用**分支管理**策略，每个分支对应特定的开发环境：

- 点击仓库顶部的分支选择器（通常显示为 `Main`）
- 浏览分支列表，找到您需要的环境名称（如 `NeoForge`、`Forge` 等）

### 2. 下载资源

进入对应分支后：

- 查看该分支的 `README.md` 了解环境详情和使用说明
- 在文件列表中找到预构建的包（通常为 `.7z` 的压缩文件）
- 点击下载
